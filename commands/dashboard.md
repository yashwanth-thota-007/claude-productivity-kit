# Dashboard

Opens the Claude Code productivity dashboard in the browser.

The dashboard visualizes:
- Discernment score trends (product, process, performance)
- Score summary cards with averages and trends
- Session timeline with effort badges and domain keywords
- Knowledge base statistics and recent facts
- Recent session replays
- Weakest dimension analysis

## Usage

```bash
python3 ~/.claude/scripts/dashboard.py
```

This command generates `~/.claude/dashboard.html` and opens it in the default browser.
